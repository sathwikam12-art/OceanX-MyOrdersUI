# MyOrdersUI - Android Assignment

A modern Android UI assignment developed using Kotlin + XML for the OceanX Agency Mobile Application Developer Internship.

## Features
- Clean and modern UI
- RecyclerView order listing
- Bottom Navigation
- Search Bar
- Filter & Sort Buttons
- Responsive Layout
- Custom Order Cards
- Clickable Tabs & Buttons

## Tech Stack
- Kotlin
- XML
- RecyclerView
- CardView
- Material Design
